# @extenzo/plugin-vue

[中文](README-zh_CN.md) | English

---

Rsbuild plugin: enables Vue 3 (SFC, Vue JSX, Babel) for extenzo projects. Use in exo.config as `plugins: [vue()]`.

- Users: `import vue from "@extenzo/plugin-vue"; plugins: [vue()]`
- Depends on @rsbuild/plugin-vue, @rsbuild/plugin-vue-jsx, @rsbuild/plugin-babel.
