# React

在 `exo.config.ts` 的 **plugins** 中接入 `@rsbuild/plugin-react` 即可启用 React 与 JSX/TSX 支持。

## 安装与配置

<Tabs>
  <Tab label="pnpm">

```bash
pnpm add react react-dom @rsbuild/plugin-react -D
```

  </Tab>
  <Tab label="npm">

```bash
npm install react react-dom @rsbuild/plugin-react -D
```

  </Tab>
  <Tab label="yarn">

```bash
yarn add react react-dom @rsbuild/plugin-react -D
```

  </Tab>
  <Tab label="bun">

```bash
bun add react react-dom @rsbuild/plugin-react -D
```

  </Tab>
</Tabs>

```ts
// exo.config.ts
import { defineConfig } from "extenzo";
import { pluginReact } from "@rsbuild/plugin-react";

export default defineConfig({
  plugins: [pluginReact()],
});
```

## 说明

- **@rsbuild/plugin-react**：为 React、JSX/TSX 提供 Rsbuild 支持，直接使用即可。
- 与 Vue、Preact、Svelte、Solid 等互斥（同一项目一般只选一个 UI 框架），与 Less/Sass 等可混用。
